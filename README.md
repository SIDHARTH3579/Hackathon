# NeuroLearn - Adaptive Mastery Learning Platform

## Prerequisites
- Java 17+
- Maven 3.6+
- Node.js 18+

## Setup & Run

### Backend
1. Navigate to `backend/`
2. Run `mvn spring-boot:run`
3. The server will start on `http://localhost:8080`

### Frontend
1. Navigate to `frontend/`
2. Run `npm install` (if not already done)
3. Run `npm run dev`
4. Open the URL shown (usually `http://localhost:5173`)

## Features
- **Adaptive Learning**: Difficulty changes based on your performance.
- **Mastery Tracking**: Visual progress bars for each topic.
- **Offline Capable**: (Basic structure implemented).
- **Gamified**: Score and progress tracking.

## Tech Stack
- **Backend**: Spring Boot 3.2, Java 17, JSON File Storage
- **Frontend**: React (Vite), Glassmorphism UI
